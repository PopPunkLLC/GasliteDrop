Created by [@PopPunkOnChain](https://twitter.com/PopPunkOnChain) and [@backseats](https://x.com/backseats_eth)

# Installation

- Clone the repo
- Install dependencies `yarn`
- Check out `.env.example`
- Run the server `yarn dev`
- Have fun!
